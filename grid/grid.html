CSS Grid

CSS grid can be initialized using display grid.

There are some properties for parent and child.

Grid properties for a parent

  1. grid-gap(defaults to row and column gap)
  2. Grid-template-columns(provides gap between columns)
  3. Grid-template-rows (provides gap between rows)
  4. Justify-content(aligns in x axis)
  5. align-content (aligns in x axis)
  6. grid-template-areas

Grid Properties for a children

  1. Justify-self (aligns in x axis)
  2. align-self (aligns in x axis)
  3. grid-column (can span in columns eg : 1/3)
  4. grid-row (can span in row eg : 1/3)
  5. grid-area(should be used with grid-template-areas property )

We can also use grid template areas to define the layout of the page
