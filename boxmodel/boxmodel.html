CSS Box model

Setting of width and height

we can set width and heigh of the component by height and width property

Setting of margin and padding

We can set margin and padding of any element with the help of margina nd padding property.

It can take upto 4 values in the order top ,right,bottom and left.

If you give two values it will be (top and bottom) then (left and right).

Setting of borders

border take 3 values they are border-width,border-style and border-color

Border-radius

We can set border radius to create rounded borders

Margin-Collapse

When two margin from different element overlap.The equivalent margin is greater of the two.This is called margin-collapse

Box-Sizing

It determines what out of padding and width and height

It has 2 values.They are

  1. Border-box
  2. Content-box

Suppose you have given width=200px .Total will be 200px in border-box.In content-box it will be 200px + padding + border + margin .
