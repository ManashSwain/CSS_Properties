CSS Flex Box

Aims at providing a better way to layout align and distribute space among items in a container.

Flex direction property

It defines the direction towards which items are laid.

  1. Row
  2. Row-reverse
  3. Column
  4. Column-reverse

We have flex properties for parent and child.

Flex properties for a parent

  1. Flex-wrap
  2. Justify-content
  3. Align-items
  4. Align-content

Flex-wrap

It can be wrap,no-wrap and wrap-reverse

Justify-content

It defines the alignment along the main axis of the web page.Some of the values are center, flex-start and flex-end.

Align-items

It defines the alignment along the cross axis. Some of the values are Cemter,flex-start and flex-end.

Align-content

Aligns a flex-container lines when there is extra space in cross axis.Some of the values are center ,flex-start and flex-end.

Flex-properties for the children

  1. order
  2. align-self
  3. flex-grow
  4. flex-shrink
  5. flex-basis

Order

Controls the order in which the items appear in the flex container.

Some of the values are 0,1,2,3....etc.The default value is 0 .If you give the order 1 then it will show the item in the last.

Align-self

Allows default alignment to be overridden for individual flex items.We have to disable align content in the container.

It is applicable for cross axis.

Some of the values are center,flex-start and flex-end.

Flex-grow

Defines the ability for a flex item to grow.Some of the values are 1,2,3,4.....etc

Default value is 0 .You need to set the value as 1 to grow.

Flex-shrink

It specifies how much a flex item will shrink relative to the rest of the flex item.

The default value is 1.You need to set as 2 to see the shrinking.

Flex-basis

When flex-direction is set to row .Flex-basis will control width.

When flex direction is set to column flex basis will control height.

Row = width

column = height

Flex- basis defines the initial width in the direction of the main axis of the child

For example : If a flex basis is set to 25 % and if there are 8 boxes . It will split into two rows.
