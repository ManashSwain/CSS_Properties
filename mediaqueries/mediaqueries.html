Media queries in CSS

Media queries in CSS are used to apply different styles to different devices or screen sizes. This allows you to create a responsive design that adapts to various screen resolutions and device capabilitie

width and height: The width and height of the viewport or the device.

min-width and max-width: Minimum and maximum width of the viewport or the device.

Preferable breakpoints

  • 640px @media (min-width: 640px) { ... }
  • 768px @media (min-width: 768px) { ... }
  • 1024px @media (min-width: 1024px) { ... }
  • 1280px @media (min-width: 1280px) { ... }
  • 1536px @media (min-width: 1536px) { ... }

Note:Reduce screen width to see the effect
