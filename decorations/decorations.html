Decoration,fonts and other CSS properties

Text-align property

It is used to set the horizontal alignment of a text

Text-decoration property

It is used to decorate the text

Text-transform property

It is used to transform the text

Some of the values are capitalize , lowercase,none and uppercase

Line height property

It is used to specify the space between lines

Fonts

Fonts play a very important role in the look and feel of a website

Font-family

Font family specifies the font of a text.

Web safe fonts

These fonts are universally installed accross browsers.

How to add google fonts ?

In order to use custom google fonts , go to google fonts then select a style and paste to the style.css of your page.

Other font properties

Some of the other font properties are listed below .

Font-size

set the size of the font.

we can set large,larger,medium,small and smaller fonts

Font-Style

Sets the font style.can be italic,normal ,oblique or oblique 40 deg.

Font-variant

Sets whether text is displayed in small-caps. Some of the values are normal and small caps.Google it to get more values.

Font-weight.

Sets the weight of the font.

Range is between (100-900).

Normal(400),bold(700) ,bolder(Bolder than the parent value) and lighther(Lighter than the parent value).

If we have imported google fonts it only supports some of the font weights .use VS code fonts so that you can use any value .

If font weights are not working try bold,bolder and ligher values

Generic families

Broad class of similar fonts .Just like when we say fruit,it can be any fruit.

Font family - specific

Generic family - generic

Box shadow

Box shadow can hold upto 5 values .They are

  1. 1st value gives shadow in the right side of the box.
  2. 2nd value gives shadow to the bottom of the box.
  3. 3rd value gives blur to the box
  4. 4th value is spread (Increase or decrease shadow width )
  5. 5th value is color

Use inset for interior shadow.

Use negative values to change the shadow of right and bottom to left and top values .

Text-Shadow

Text-shadow is none for default

The properties are same as of box shadow .

If we want multiple shadows we can put comma and repeat.
