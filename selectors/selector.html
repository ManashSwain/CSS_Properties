CSS Selectors and Colors

CSS Selector are used to find the element whose property will be set.

Selectores are used to target the HTML elements.

Selectors makes it easy for us to easily target single/Multiple HTML elements in the markup.

Types of basic CSS selector

  1. CSS element selector
  2. CSS Id selector
  3. CSS class selector
  4. CSS grouping selector
  5. CSS Universal selector

Colors in CSS

RGB

Specify colors using Red, Green and Blue values

Limit : 0-255 (0,0,0 - black) and (255,255,255 - white)

HEX code

Specify using HEX code.Example:#ff7180

ff denotes red , 71 denotes green and 80 denotes blue

HSL Values

Specify the colors using HSL values (Hue ,saturation and Lightness).Example : hsl(89,67%,67%)

hue is degress (min : 0 and max 360 degree).stauration and lightness are in percentage (min: 0% and max:100%)
