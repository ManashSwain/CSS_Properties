CSS Transforms

Transforms are used to rotate,move,skew and scale elements.They are used to create a 3d effects.

The transform property

It is used to apply a 2D or 3D transformation to an element.

The Transform-origin-property

allows to change the position of transformed elements

  1. 2D transforms (can change x and y axis)
  2. 3D transforms (can change z axis as well)

CSS 2D transform methods

We can use the following 2D tranforms in CSS

  1. translate()
  2. rotate()
  3. scaleX()
  4. scaleY()
  5. skew()
  6. matrix() (It takes 6 values in it)
  7. scale()

CSS 3D transform methods

  1. rotateX()
  2. rotateY()
  3. rotateZ()
