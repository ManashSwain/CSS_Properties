CSS (Cascading Style Sheets)

HTML is just an skeletal layout of a website.

We need CSS to design a website and add styles to it and make it look beautiful

Cascading Style Sheets known as CSS is a style sheet language that is used to handle the presentation of the webpage containing HTML

It makes our Website beautiful and modern looking

DOM(Document Object Model)

DOM stands for Document Object Model when a page is loaded, the browser creates a DOM of the page which is constructed as a tree of object

Three ways to add CSS to HTML

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Inline CSS

CSS is added to the elements directly using style attribute

Internal CSS

CSS is kept inside the head of HTML inside the style tag

External CSS

Css is kept inside a .CSS style sheet

Priority of selectors

The priority of inline selector is greater then internal selector and the priority of internal selector is greater then external selector.
